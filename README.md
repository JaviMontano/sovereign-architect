# Sovereign Architect

> Liderazgo técnico basado en evidencia

Plugin de arquitectura de software que actúa como Principal Software Systems Lead. Analiza código, repositorios, especificaciones e incidentes para decisiones arquitectónicas fundamentadas.

---

## 🚀 Instalación Rápida

```bash
git clone https://github.com/JaviMontano/mao-sovereign-architect.git ~/.claude/plugins/mao-sovereign-architect
```

## 🔑 Qué Hace

- 66 agentes de arquitectura de software
- Análisis de deuda técnica con cuantificación
- ADR automatizados (Architecture Decision Records)
- Modelado C4 y diagramas Mermaid
- Auditoría de seguridad y rendimiento
- Code review multi-dimensional

## 📐 Arquitectura

```
principal-architect orquesta → 66 agentes especialistas → evidence-auditor valida
```

# Sovereign Architect

> Evidence-based technical leadership

Software architecture plugin that acts as Principal Software Systems Lead. Analyzes code, repositories, specifications, and incidents for evidence-based architectural decisions.

### What It Does

- 66 software architecture agents
- Technical debt analysis with quantification
- Automated ADRs (Architecture Decision Records)
- C4 modeling and Mermaid diagrams
- Security and performance auditing
- Multi-dimensional code review

### Architecture

```
principal-architect orchestrates → 66 specialist agents → evidence-auditor validates
```
